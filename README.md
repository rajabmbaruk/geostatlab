3. 🧾 Extended Abstract 
GeoStatLab is an interactive, computer-assisted teaching platform designed to enhance learning in spatial statistics and national statistical systems. The tool simulates real-world workflows used by National Statistical Offices (NSOs), enabling users to design surveys, generate and analyze socio-economic datasets, and visualize spatial patterns using GIS-based techniques.
The platform is built using realistic datasets inspired by operations of the Kenya National Bureau of Statistics (KNBS), including household surveys, population data, and agricultural indicators. GeoStatLab integrates statistical concepts such as sampling design, data cleaning, and descriptive analysis with geospatial visualization, allowing learners to explore how data informs public policy decisions.
The system consists of four core modules: (i) a survey simulation module that allows users to design sampling strategies and observe their effects; (ii) a spatial analysis module for mapping and visualizing regional disparities; (iii) a data analytics dashboard for interactive exploration of key indicators; and (iv) a policy simulation module that demonstrates the impact of interventions on socio-economic outcomes.
GeoStatLab addresses a critical gap in statistics education by bridging theoretical learning with applied, real-world data workflows. It is particularly relevant for universities, training institutions, and NSOs in developing countries, where practical exposure to integrated statistical and geospatial systems is limited.
The platform contributes to capacity building in spatial data science and supports the integration of statistical and geospatial information for evidence-based decision-making. It aligns with global priorities in Earth observation, sustainable development, and data-driven governance.

4. 🎯 Objectives
The objectives of GeoStatLab are to:
•	Enhance teaching of spatial statistics through interactive learning 
•	Simulate real-world NSO workflows (survey design → analysis → policy use) 
•	Improve understanding of geospatial data integration 
•	Support capacity building in developing countries 
•	Promote data-driven decision-making skills 

5. 🧩 Description of the Software
GeoStatLab is a web-based interactive application developed to support teaching and training in statistics and geospatial analysis.
Key Functional Components:
1. Survey Simulation Module
•	Users define: 
o	Population characteristics 
o	Sampling method (random, stratified, cluster) 
•	System generates synthetic datasets 
•	Demonstrates sampling bias and errors 
2. Spatial Analysis Module
•	Interactive map interface 
•	Displays indicators such as: 
o	Population density 
o	Poverty levels 
o	Agricultural productivity 
•	Supports choropleth mapping and regional comparisons 
3. Data Analysis Dashboard
•	Interactive charts and tables 
•	Users perform: 
o	Descriptive statistics 
o	Trend analysis 
•	Enables comparison across regions 
4. Policy Simulation Module
•	Scenario-based analysis 
•	Users simulate interventions (e.g., subsidies, resource allocation) 
•	Outputs projected socio-economic impacts 

6. 🧠 Innovation & Educational Value
GeoStatLab introduces several innovations:
•	Integration of statistics + GIS + policy simulation in one platform 
•	Use of realistic NSO workflows rather than abstract datasets 
•	Focus on applied learning in developing country contexts 
•	Interactive simulation of decision-making processes 
Educational benefits include:
•	Improved conceptual understanding through visualization 
•	Hands-on learning of survey and spatial analysis techniques 
•	Bridging theory and real-world application
